# CodeSikho - AI for Bharat Hackathon 2026

Multilingual AI Learning Platform for India's 50M+ Coding Students


## Problem

90% of programming resources are in English, creating a barrier for India's 300M+ non-English speaking students.


## Solution

AI-powered coding mentor that explains errors and concepts in 7 Indian languages with voice support.


## Key Features

Multilingual support (Hindi, Tamil, Telugu, Bengali, Kannada, Malayalam, English)

Voice-first interaction (Amazon Transcribe + Polly)

Context-aware AI (Amazon Bedrock - Claude Sonnet)

Practice problem generator

Personalized learning paths

Offline-capable PWA


## AWS Services

Bedrock, Transcribe, Polly, Lambda, ECS, RDS, DynamoDB, S3, CloudFront, Cognito, API Gateway, Route 53, WAF, Secrets Manager, CloudWatch, X-Ray, ElastiCache, SES

Total: 18 AWS Services
